CSS Gradients

CSS gradients is used to display transitions between two or more colors, it requires at least 2 color stops.

There are 3 types of gradients in CSS:
- Linear Gradients (goes down/up/left/right/diagonally)
- Radial Gradients (defined by their center)
- Conic Gradients (rotated around a center point)

Syntax: background: linear-gradient(direction, "color1 stop" , "color2 stop", ..., "color-n stop");

direction:
- keyword values: to right, to left, to bottom(default), to top, to bottom left/right, to top left/right.
- deg values: 90deg, -90deg, 180deg, 0 deg...
- turn units: 0.33turn, 0.25 turn...

stop:
- represents the color stop point at which that color will stop spanning and change to the next color stop.
- can be %, px or other units.
- if stops are not provided, all colors will take an equal area to span.

Basic CSS patterns

Two-color symmetric gradient:
background: linear-gradient(pink, purple);

Color stop points are not specified so two colors took an equal area and met at the center of the element.
Two-color asymmetric gradient:
background: linear-gradient(pink 80%, purple);

Pink color spans from 0 to 80% of the elem and purple spans from 80% to 100%
Two-color solid pattern:
background: linear-gradient(pink 60%, purple 60%);

If two color stop points are provided equally, we will get a solid color combination.

Pink color spans from 0 to 60% of the elem and purple spans from 60% to 100%
Two-color directional pattern:
background: linear-gradient(to right, pink , purple );
Multiple-color pattern:
background: linear-gradient(0.9turn, orange, pink, purple, rgb(23, 189, 167));

Intermediate patterns

Vertical bars:
background-color: purple;
background-image: linear-gradient( 90deg, transparent 50%, rgba(255, 255, 255, 0.5) 50% );

The 1st half shows original color, the 2nd half shows 50% transparent.
Repeating vertical bars:
background-color: purple;
background-image: linear-gradient( 90deg, transparent 50%, rgba(255, 255, 255, 0.5) 50% );
background-size: 20px;

The 1st half shows original color, the 2nd half shows 50% transparent.
Checkerboard pattern :
background-color: purple;
background-image:
linear-gradient( transparent 50%, rgba(255, 255, 255, 0.5) 50% ),
linear-gradient(90deg, transparent 50%, rgba(255, 255, 255, 0.5) 50%);
Checkerboard pattern :
background-color: purple;
background-image:
linear-gradient( transparent 50%, rgba(255, 255, 255, 0.5) 50% ),
linear-gradient(90deg, transparent 50%, rgba(255, 255, 255, 0.5) 50%);
background-size: 20px 20px;

background-size: is used to set size of background-image.

Advanced patterns

Using multiple linear-gradients and overlay them on top of one another to create the shape.

Diamond pattern :
background-color: purple;
background-image:
linear-gradient(135deg, pink 25%, transparent 25%),
linear-gradient(225deg, pink 25%, transparent 25%),
linear-gradient(45deg, pink 25%, transparent 25%),
linear-gradient(315deg, pink 25%, transparent 25%);


Create each side of the diamond based on a specific angle
