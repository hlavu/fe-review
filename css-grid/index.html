Positioning items

To make an element span horizontally from the left to the right using:
grid-colum-start: the n column line;
grid-colum-end: the n+x column line;
short hand: grid-column: start/end OR start / span numberOfColumns;

To make an element span vertically from the top to the bottom using:
grid-row-start: the n row line;
grid-row-end: the n+x row line;
short hand: grid-row: start/end OR start / span numberOfRows;

The number of column/row lines = number of columns/rows + 1;
The first column/row line has index of 1
The last column/row line has index of number of col/row + 1 OR -1

Header grid-column: 2/-1
Menu grid-row: 1/-2
Content grid-column: 2/-1
Footer grid-column: 1/-1

Header grid-column: 1/-2
Menu grid-row: 1/-2 grid-column: -1/-2
Content grid-column: 1/-2
Footer grid-column: 1/-1
